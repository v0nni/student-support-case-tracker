# student-support-case-tracker
A small system schools or nonprofits could use to track student support needs (academic, behavioral, health, accommodations).  This ties naturally into:  Case Management  Accessibility / inclusion  Workflow + automation  Clean data modeling

## Student Support Case Tracker (Salesforce)

### Overview
A lightweight Salesforce application for tracking student support needs such as
academic assistance, health concerns, and accessibility accommodations.

### Features
- Custom Student Support object
- Flow automation for priority assignment
- Lightning Web Component embedded on record pages
- Reports and dashboards for visibility

### Why This Project
Built to demonstrate:
- Salesforce data modeling
- Declarative automation
- UI customization
- Real-world problem solving

### Tech Used
- Salesforce Lightning Experience
- Flow Builder
- LWC / Visualforce
- Reports & Dashboards
